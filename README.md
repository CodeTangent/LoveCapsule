# 💌 Love Capsule

**Love Capsule** é uma solução digital criada para transformar mensagens em uma **experiência interativa e personalizada**.

A proposta é simples: ao invés de enviar apenas uma mensagem, você entrega uma **página única**, com fotos, música e momentos especiais — acessível via link ou QR Code.

---

## 🔗 Demonstração

👉 **Ver demo:**

## 🎯 Proposta

Criar uma forma moderna, emocional e memorável de presentear alguém, utilizando tecnologia para potencializar conexões humanas.

Não é apenas um site, é um **produto digital personalizado**.

---

## ✨ Funcionalidades

Cada cápsula entrega uma jornada interativa:

* 💬 **Mensagem personalizada**
* 📅 **Linha do tempo do relacionamento**
* 🖼 **Galeria de fotos**
* 🎵 **Música especial do casal**
* 💭 **Memórias aleatórias (efeito surpresa)**
* 📱 **QR Code para compartilhamento rápido**

---

## 🧠 Estrutura do projeto

O projeto foi desenvolvido com foco em **reutilização e escalabilidade**, separando layout e conteúdo.

Estrutura base:

```text
index.html   → Estrutura fixa da página
dados.js     → Conteúdo personalizado do cliente
img/         → Fotos
audio/       → Música
qr_codes/    → Geração de QR Code (Python)
```

✔ Criação rápida de novas cápsulas  
✔ Baixo retrabalho  
✔ Fácil manutenção
